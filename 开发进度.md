# DataSay 开发进度跟踪

## 项目概览
DataSay 是一个企业级多模态内容创作平台，支持文本、语音、图像/视频的智能生成与管理。

### 技术栈
- **后端**: FastAPI + PostgreSQL + Redis + Celery
- **前端**: React 18 + TypeScript + Ant Design
- **AI集成**: 火山引擎（豆包）、Azure、OpenAI
